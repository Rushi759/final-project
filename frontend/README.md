# 🌿 AgroGuru - Agricultural Assistant Platform

AgroGuru is a feature-rich React-based web application designed to empower farmers with data-driven insights and easy access to agricultural resources.

## ✨ Key Features

### 🚜 Resource Locators
- **Plant Nurseries:** Locate nearby nurseries using real-time geolocation. View available plants and contact owners directly.
- **Agricultural Markets:** Stay updated with crop prices and market locations.
- **Testing Laboratories:** Find soil and plant testing labs within a 50km radius.

### 📚 Expert Tools
- **Crop Encyclopedia:** Searchable database with 22+ crops, including detailed info on temperature, moisture, and soil requirements.
- **Seasonal Calendar:** Visual timeline for Kharif, Rabi, and Zaid seasons.
- **Fertilizer Calculator:** Instant NPK requirement calculation for various land units (Acre, Hectare, Bigha).

### 🎨 Modern Design
- **Premium UI:** Glassmorphism aesthetics on all major interface elements.
- **Dark Mode:** Seamless theme switching for night-time usage.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.

## 🏗️ Technical Stack

- **Frontend:** React.js, React Router, React Hook Form, Axios, Yup.
- **Backend:** Node.js, Express.js, JWT Authentication.
- **Database:** MongoDB Atlas (NoSQL).
- **Styling:** Vanilla CSS3 with Custom Design System.

## 🚀 Setup & Installation

### 1. Backend Setup
1. Navigate to the `backend` directory.
2. Install dependencies: `npm install`
3. Configure `.env` with your **MONGODB_URI** and **JWT_SECRET**.
4. Start the server: `npm run dev`

### 2. Frontend Setup
1. Navigate to the root directory.
2. Install dependencies: `npm install`
3. Add `"proxy": "http://localhost:8080"` to your `package.json`.
4. Start the app: `npm start`
