## Project Overview
AgroGuru is a smart agricultural intelligence platform designed to bridge the gap between farmers and essential services like nurseries, laboratories, and markets. It features a regional-first approach, prioritizing local services in the Satara, Sangli, and Kolhapur districts of Maharashtra.
